# LaTeX
## Installation
 - #### Ubuntu
   ```$ sudo apt-get install texlive-full```
   ```$ sudo apt-get install texmaker```
 - #### Windows
    Download  and Install [MikTeX](https://miktex.org/download) and [TeXstudio](http://www.texstudio.org/)
